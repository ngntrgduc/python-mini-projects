### Requirements

```py
pip install fire pydub PyAudio
```

### Usage
- Normal usage, with 25 minutes
```py
python pomodoro.py
```
- Custom minutes
```py
python pomodoro.py <minutes>
```
Example:
```py
python pomodoro.py 40
```
```py
python pomodoro.py 5
```
- Flowtime
```py
python pomodoro.py -f
```
or 
```py
python pomodoro.py --flow
```
- Get some help
```py
python pomodoro.py -h
```
